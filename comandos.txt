# como vincular um repositorio do GOTHUb com o visual code
1 no GITHUB abra o repositorio e clique no botão CODE -> Local -> copiar link do repositorio
2 No visual Code abra a pasta de seus projetos ( exemplo: Documentos/Python/ )
3 Abra o terminal e ite o comando: git clone <clone aqui o link copiado>
4 tecle ENTER
5 No VS Code Abra a pasta clonada 
6 No Terminal digite: git config user.name <digite aqui seu nome de usúario no github>
7 Tecle ENTER
8 Ainda no terminal digite: git config user.email <digite aqui seu email no GitHub>
Teclen ENTER
